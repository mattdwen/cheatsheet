# Cheatsheet

## Editors

Atom, Visual Studio, and VS Code.

- **Find in Files**

  `ctrl+shift+f`

- **Markdown Preview**

  `ctrl+shift+m`

  *Note: Custom binding in VS Code.*

## Git

- **Force Push**

  ```
  git push origin master --force
  ```

- **Rebase**

  ```
  git checkout experiment
  git rebase master
  ```

## Visual Studio

- **Search Solution Explorer**

  `ctrl+;`
