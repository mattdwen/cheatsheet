# Cheatsheet

## Editors

Atom, Visual Studio, and VS Code.

- **Find in Files**

  `ctrl+shift+f`

## Git

- **Force Push**

  ```
  git push origin master --force
  ```

- **Rebase**

  ```
  git checkout experiment
  git rebase master
  ```

## Visual Studio Code

- **Open Markdown Preview**

    `ctrl+k` `v`
